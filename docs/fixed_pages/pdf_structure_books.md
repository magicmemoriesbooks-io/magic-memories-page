# Estructura PDF - Libros Personalizados y Haz tu Historia

## Formato
- **Tamaño interior**: 210 x 297 mm (A4 vertical)
- **Resolución**: 300 DPI
- **Píxeles**: 2480 x 3508 px

## Especificaciones Lulu (Hardcover A4)

### Cubierta (archivo separado)
- **Tipo**: Spread completo (contraportada + lomo + portada)
- **Ancho total**: Contraportada (216mm) + Lomo (variable) + Portada (216mm)
- **Alto**: 303mm (incluye sangrado 3mm arriba y abajo)
- **Cálculo del lomo**: 0.0572mm × número de páginas + 1.52mm

### Interior
- **Sangrado**: 3mm en todos los lados
- **Márgenes seguros**: 15mm del borde (sin texto importante)

## Estructura de Páginas

### PDF Digital (23 páginas)
| Página | Contenido |
|--------|-----------|
| 1 | Portada interior (título + ilustración portada) |
| 2 | Dedicatoria |
| 3-21 | Páginas de historia (19 escenas con texto) |
| 22 | Página final épica ("Fin" / "The End") - SIN texto narrativo |
| 23 | Créditos |

### PDF Imprimible Lulu (24 páginas)
| Página | Contenido |
|--------|-----------|
| 1 | Portada interior |
| 2 | Página en blanco (reverso portada) |
| 3 | Dedicatoria |
| 4-22 | Páginas de historia (19 escenas) |
| 23 | Página final épica |
| 24 | Créditos |

**Nota**: La cubierta (portada + contraportada + lomo) es un archivo PDF separado.

## Tipografía

### Para todas las edades (3-7 años)
- **Fuente título**: Decorativa/Fantasy, 48-60pt
- **Fuente texto**: Serif legible (Georgia, Crimson Text), 18-22pt
- **Interlineado**: 1.5
- **Color texto**: Blanco con sombra negra sutil

## Ubicación del Texto en Ilustraciones

### Opción preferida: Banda inferior
- **Altura banda**: 15-20% de la imagen
- **Fondo**: Degradado semi-transparente (negro 70% → transparente)
- **Padding**: 30px horizontal, 20px vertical

### Opción alternativa: Banda superior
- Para escenas donde la acción está en la parte inferior
- Mismas especificaciones que banda inferior

## Página Final Épica (Página 22)
- Ilustración a página completa
- Solo texto "Fin" / "The End" centrado
- Tipografía decorativa grande (72-96pt)
- Sin texto narrativo

## Watermarks (Pre-pago)
- Texto diagonal: "Magic Memories Books"
- Opacidad: 30%
- Rotación: 45°
- Se aplica a TODAS las ilustraciones
- Se elimina después del pago

## Post-Pago Processing
1. Upscaling a 300 DPI (3508 x 2480 px)
2. Overlay de texto sobre ilustraciones
3. Generación PDF sin watermarks
4. Envío automático a Lulu
